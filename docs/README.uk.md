# WebServ

Сучасний кросплатформовий настільний додаток для локальної веб-розробки.

## Опис

WebServ — це настільний додаток, розроблений для спрощення локальної веб-розробки шляхом надання сучасного кросплатформового рішення. Він використовує Electron для забезпечення нативного настільного досвіду, React для динамічного користувацького інтерфейсу та інтегрується з Docker через `dockerode` для управління середовищами локальної веб-розробки. Цей інструмент має на меті оптимізувати налаштування та управління різними веб-сервісами та проектами на вашій локальній машині.

## Можливості

*   **Кросплатформенність**: Доступно для Windows, macOS та Linux.
*   **Управління середовищем локальної веб-розробки**: Легко керуйте своїми веб-проектами та сервісами.
*   **Інтеграція з Docker**: Використовуйте контейнери Docker для ізольованих та узгоджених середовищ розробки.
*   **Сучасний користувацький інтерфейс**: Створено з використанням React для чуйного та інтуїтивно зрозумілого користувацького досвіду.
*   **Управління конфігурацією**: Використовує `electron-store` та `yaml` для постійних налаштувань програми.

## Використовувані технології

*   **Electron**: Для створення кросплатформових настільних додатків.
*   **React**: Бібліотека JavaScript для створення користувацьких інтерфейсів.
*   **TypeScript**: Типізована надмножина JavaScript, яка компілюється в звичайний JavaScript.
*   **Vite**: Швидкий інструмент збірки для сучасних веб-проектів.
*   **Dockerode**: Модуль Node.js для взаємодії з демоном Docker.
*   **Axios**: HTTP-клієнт на основі промісів для браузера та Node.js.
*   **Electron Log**: Простий модуль логування для Electron.
*   **Electron Store**: Зберігайте та завантажуйте дані як професіонал.
*   **YAML**: Для парсингу та перетворення YAML у рядок.

## Встановлення

Щоб отримати локальну копію та запустити її, виконайте наступні прості кроки.

### Попередні вимоги

*   Node.js (рекомендується LTS-версія)
*   npm (поставляється з Node.js)
*   Docker Desktop (або Docker Engine) встановлено та запущено

### Кроки

1.  **Клонуйте репозиторій:**
    ```bash
    git clone https://github.com/mZer0ne/WebServ.git
    cd WebServ
    ```
2.  **Встановіть залежності:**
    ```bash
    npm install
    ```

## Використання

### Режим розробки

Щоб запустити додаток у режимі розробки:

```bash
npm run dev
npm run electron:dev
```

Це запустить сервер розробки Vite, а потім запустить додаток Electron, що дозволить використовувати гарячу перезавантаження та спростить налагодження.

### Збірка для продакшену

Щоб зібрати додаток для вашої операційної системи:

*   **Для Windows (64-біт):**
    ```bash
    npm run electron:win
    ```
*   **Для macOS (ARM64):**
    ```bash
    npm run electron:mac
    ```

Результат збірки буде знаходитись у каталозі `release`.
