# WebServ

Современное кроссплатформенное настольное приложение для локальной веб-разработки.

## Описание

WebServ — это настольное приложение, разработанное для упрощения локальной веб-разработки путем предоставления современного кроссплатформенного решения. Оно использует Electron для обеспечения нативного настольного опыта, React для динамического пользовательского интерфейса и интегрируется с Docker через `dockerode` для управления средами локальной веб-разработки. Этот инструмент призван оптимизировать настройку и управление различными веб-сервисами и проектами на вашей локальной машине.

## Возможности

*   **Кроссплатформенность**: Доступно для Windows, macOS и Linux.
*   **Управление средой локальной веб-разработки**: Легко управляйте своими веб-проектами и сервисами.
*   **Интеграция с Docker**: Используйте контейнеры Docker для изолированных и согласованных сред разработки.
*   **Современный пользовательский интерфейс**: Создан с использованием React для отзывчивого и интуитивно понятного пользовательского опыта.
*   **Управление конфигурацией**: Использует `electron-store` и `yaml` для постоянных настроек приложения.

## Используемые технологии

*   **Electron**: Для создания кроссплатформенных настольных приложений.
*   **React**: Библиотека JavaScript для создания пользовательских интерфейсов.
*   **TypeScript**: Типизированный надмножество JavaScript, которое компилируется в обычный JavaScript.
*   **Vite**: Быстрый инструмент сборки для современных веб-проектов.
*   **Dockerode**: Модуль Node.js для взаимодействия с демоном Docker.
*   **Axios**: HTTP-клиент на основе промисов для браузера и Node.js.
*   **Electron Log**: Простой модуль логирования для Electron.
*   **Electron Store**: Сохраняйте и загружайте данные как профессионал.
*   **YAML**: Для парсинга и преобразования YAML в строку.

## Установка

Чтобы получить локальную копию и запустить ее, выполните следующие простые шаги.

### Предварительные требования

*   Node.js (рекомендуется LTS-версия)
*   npm (поставляется с Node.js)
*   Docker Desktop (или Docker Engine) установлен и запущен

### Шаги

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/mZer0ne/WebServ.git
    cd WebServ
    ```
2.  **Установите зависимости:**
    ```bash
    npm install
    ```

## Использование

### Режим разработки

Чтобы запустить приложение в режиме разработки:

```bash
npm run dev
npm run electron:dev
```

Это запустит сервер разработки Vite, а затем запустит приложение Electron, что позволит использовать горячую перезагрузку и упростит отладку.

### Сборка для продакшена

Чтобы собрать приложение для вашей операционной системы:

*   **Для Windows (64-бит):**
    ```bash
    npm run electron:win
    ```
*   **Для macOS (ARM64):**
    ```bash
    npm run electron:mac
    ```

Результат сборки будет находиться в каталоге `release`.
